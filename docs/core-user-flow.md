# Core User Flow

## MVP Flow

1. User opens the dashboard.
2. User reviews demo detected companies.
3. User inspects company risk level and explanation.
4. User creates a draft GDPR-style request.
5. User tracks request status.

## Future Production Flow

1. User creates and verifies an account.
2. User manually adds or safely discovers organisations.
3. System calculates transparent risk indicators.
4. User selects a controlled request template.
5. User reviews the request before sending.
6. System records structured audit events.
7. User tracks status and can delete retained data.
